# 📊 Source of Information Dashboard

## 📌 Overview

This project was developed as **Task 6** of the **Cognifyz Technologies Power BI Internship**. The dashboard analyzes the various sources from which investors gather investment-related information and identifies the most trusted and commonly used sources.

---

## 🎯 Objective

The objective of this dashboard is to:

- Analyze the different sources of investment information.
- Identify the most preferred information source among investors.
- Compare information sources with investment avenues.
- Present meaningful insights using interactive Power BI visualizations.

---

## 📂 Dataset

The dataset contains investor-related information, including:

- Gender
- Investment Avenue
- Source of Investment Information
- Savings Objectives
- Investment Duration
- Monitoring Frequency
- Investment Reasons

---

## 📈 Dashboard Features

### 📌 KPI Cards
- Total Investors
- Total Information Sources
- Most Preferred Information Source

### 📊 Information Source Distribution
Displays the frequency of different investment information sources.

### 📊 Information Source vs Investment Avenue
Compares investment avenues based on the source of investment information.

### 🥧 Information Source Percentage
Shows the percentage contribution of each information source.

### 🎛 Interactive Filters
- Gender
- Investment Avenue

---

## 🔍 Key Insights

- The Internet is the most preferred source of investment information.
- Newspapers and financial advisors remain trusted information sources.
- Different investment avenues are influenced by different information sources.
- Interactive filters help analyze investment behavior across demographics.
